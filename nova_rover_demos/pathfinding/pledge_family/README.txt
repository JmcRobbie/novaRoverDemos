This folder contains all files required for running a modified version of the Pledge pathfinding algorithm.


Scripts included:

> modified_pledge | this is the master script, which contains the minimal function call of the algorithm. Requires import of mouse.py and labyrinth.py

> mouse.py	  | this script contains the logic behind the algorithm itself	

> labyrinth.py    | this script details how the algorithm processes and interacts with the maze environment

> demo.py and demo_labyrinth.py | These scripts are a demonstration of the use of the algorithm, including visualising the solution to the demonstration maze.
